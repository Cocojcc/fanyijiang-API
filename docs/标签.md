## 标签

### 新建标签

**请求API**

+ `/api/labels/`

**请求方式**

+ POST

**请求头参数**

| 参数名        | 必选 | 类型   | 说明                  |
| :------------ | :--- | :----- | :-------------------- |
| authorization | 是   | string | token值，用于用户校验 |

**参数**

| 参数名 | 必选 | 类型   | 说明     |
| :----- | :--- | :----- | -------- |
| name   | 是   | string | 标签名称 |
| intro  | 是   | string | 标签介绍 |

**返回示例**

```json
{
    "code": 0,
    "data": {
        "name": "数学",
        "intro": "研究数量关系与空间形式",
        "pk": 1
    }
}
```

### 获取指定标签及其子标签

**请求API**

+ `/api/labels/relations/(?P<pk>\d+)`

**请求方式**

+ GET

**URL参数**

| 参数名 | 必选 | 类型    | 说明     |
| :----- | :--- | :------ | -------- |
| pk     | 是   | integer | 父标签ID |

**返回示例**

```json
{
    "code": 0,
    "data": {
        "parent": {
            "name": "数学",
            "intro": "研究数量关系与空间形式",
            "pk": 1
        },
        "children": [
            {
                "name": "代数",
                "intro": "代数结构和运算",
                "pk": 6
            },
            {
                "name": "分析",
                "intro": "微积分的发展",
                "pk": 7
            }
        ]
    }
}
```
