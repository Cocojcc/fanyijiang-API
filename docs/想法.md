## 想法

### 发表想法

**请求API**

+ `/api/ideas/`

**请求方式**

+ POST

**请求头参数**

| 参数名        | 必选 | 类型   | 说明                  |
| :------------ | :--- | :----- | :-------------------- |
| authorization | 是   | string | token值，用于用户校验 |

**参数**

| 参数名  | 必选 | 类型   | 说明       |
| :------ | :--- | :----- | ---------- |
| content | 是   | string | 想法的内容 |

**返回示例**

```json
{
    "code": 0,
    "data": {
        "user_id": "cd2ed05828ebb648a225c35a9501b007",
        "content": "开始吧",
        "create_at": "20200228 18:12:33",
        "id": 5,
        "agree_count": 0,
        "nickname": "新手",
        "avatar": "images/001.png"
    }
}
```

+ POST

**请求头参数**

| 参数名        | 必选 | 类型   | 说明                  |
| :------------ | :--- | :----- | :-------------------- |
| authorization | 是   | string | token值，用于用户校验 |

**参数**

| 参数名 | 必选 | 类型    | 说明                             |
| :----- | :--- | :------ | -------------------------------- |
| id     | 是   | integer | 想法或评论的ID                   |
| type   | 否   | string  | idea表示想法，其他或省略表示评论 |

**返回示例**

```json
{
    "code": 0,
    "data": null
}
```
