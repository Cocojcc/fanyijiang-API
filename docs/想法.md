## 想法

### 发表想法

**请求API**

+ `/api/ideas/`

**请求方式**

+ POST

**请求头参数**

| 参数名        | 必选 | 类型   | 说明                  |
| :------------ | :--- | :----- | :-------------------- |
| authorization | 是   | string | token值，用于用户校验 |

**参数**

| 参数名  | 必选 | 类型   | 说明       |
| :------ | :--- | :----- | ---------- |
| content | 是   | string | 想法的内容 |

**返回示例**

```json
{
    "code": 0,
    "data": {
        "id": 4,
        "content": "我有一个绝妙的证明",
        "create_at": "20200319 09:44:06",
        "agree_count": 0,
        "author_info": {
            "nickname": "手机用户4zvDTn3",
            "avatar": "/avatar/36dda3061ff7738850aa522ddb900f27.jpg",
            "slug": "shou-ji-yong-hu-4zvdtn3"
        }
    }
}
```

### 想法评论点赞

**请求API**

+ `/api/ideas/likes/`

**请求方式**

+ POST

**请求头参数**

| 参数名        | 必选 | 类型   | 说明                  |
| :------------ | :--- | :----- | :-------------------- |
| authorization | 是   | string | token值，用于用户校验 |

**参数**

| 参数名 | 必选 | 类型    | 说明                             |
| :----- | :--- | :------ | -------------------------------- |
| id     | 是   | integer | 想法或评论的ID                   |
| type   | 否   | string  | idea表示想法，其他或省略表示评论 |

**返回示例**

```json
{
    "code": 0,
    "data": null
}
```

### 想法评论取消点赞

**请求API**

+ `/api/ideas/likes/?id=<id>`

**请求方式**

**请求头参数**

| 参数名        | 必选 | 类型   | 说明                  |
| :------------ | :--- | :----- | :-------------------- |
| authorization | 是   | string | token值，用于用户校验 |

**查询字符串参数**

| 参数名 | 必选 | 类型    | 说明               |
| :----- | :--- | :------ | ------------------ |
| id     | 是   | integer | 想法评论的点赞的ID |

**返回示例**

```json
{
    "code": 0,
    "data": null
}
```
